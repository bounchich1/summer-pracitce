Инструкция по сборке для Windows:

1) Откройте терминал и перейдите в корневую директорию проекта
2) Установите требуемые библиотеки с помощью команды - pip install -r requirements. txt
3) Введите команду: pyinstaller main.py
4)Перейдите в папку с exe файлом, введя команду: cd .\dist\main
5)Чтобы запустить программу необходимо ввести команду: start main.exe

Инструкция по сборке для Linux:

1) Необходимо установить Anaconda с официального сайта
2) Запустите файл install.sh
3) После окончания создания виртуальной среды и установки всех необходимых зависимостей запустите файл start.sh, в дальнейшем для запуска программы используйте этот же файл
